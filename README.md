# Buto-Plugin-FlagsLipis_6_1_1
Show flags in span tags.

## Github
```
https://github.com/lipis/flag-icons
```

## Include

```
type: widget
data:
  plugin: flags/lipis_6_1_1
  method: include
```

## Usage
Use them in span tags.
```
type: span
attribute: 
  class: fi fi-se
```
Squared version by adding fis.
```
type: span
attribute: 
  class: fi fi-se fis
```
